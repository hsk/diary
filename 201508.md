### Makefile

	all:
		php README.php > README.md
		git commit -a
		git push

こんなファイルを作っておきます。
月ごとに１つのファイルにマークダウンでメモ書きを残して行きます。
201508.mdに何か書くと。

で、

	make

とすると、README.phpが動いて、README.mdが更新され、git commitされるのでコメント書いて、git pushは自動的に行われる。
便利？良くわからないですけど。
